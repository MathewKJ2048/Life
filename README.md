# Life
Conway's Game of Life, written as a webapp using HTML5 Canvas and vanilla javascript.

Try it online here: [https://mathewkj2048.github.io/Life/](https://mathewkj2048.github.io/Life/)

![screenshot](image.png)

## Features:

- step
- pause/play
- zoom in and out
- click to add cell
- reset
- infinite scroll
